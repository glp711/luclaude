# Lume Casa

Site estático de loja de perfumaria de ambiente, criado a partir de um catálogo inicial com 243 produtos.

## Arquivos

- `index.html`: estrutura da loja
- `styles.css`: visual responsivo
- `app.js`: busca, filtros, modal de produto e carrinho demonstrativo
- `products.js`: catálogo inicial de produtos

## Como abrir

Abra `index.html` no navegador ou publique a pasta em qualquer hospedagem estática.

Para GitHub Pages, use a branch `main` e a pasta raiz do repositório.
